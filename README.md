# Pharmachrome

Pharmachrome is a clinical pharmacy calculator platform designed for pharmacy students, researchers, and healthcare professionals.

## Features

- Clinical Pharmacy Calculators
- Pharmacokinetics Calculations
- Dose Calculators
- Creatinine Clearance Calculators
- IV Infusion Calculators
- Unit Conversion Tools

## Categories

- Fundamental Calculations
- Clinical Calculations
- Pharmacokinetics Calculations
- Renal Function Calculations
- Compounding Calculations
- Nutrition Calculations

## Live Website

https://pharmachrome.github.io

## Purpose

Pharmachrome provides accurate pharmaceutical calculations based on academic and clinical references.

## Target Users

- Pharmacy Students
- Pharmacists
- Researchers
- Healthcare Professionals
